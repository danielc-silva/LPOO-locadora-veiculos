# Locadora de Veículos 🚗

Este é um projeto de aprendizagem desenvolvido para a disciplina de **Linguagem de Programação Orientada a Objetos (LPOO)**.

O objetivo deste repositório é praticar a construção de um sistema em Python aplicando os conceitos vistos em aula, evoluindo o código conforme o semestre avança.

## Declaração de Uso de IA

Declaro que houve o auxílio de ferramentas de Inteligência Artificial durante o desenvolvimento deste projeto, servindo como apoio para a estruturação do código, boas práticas e resolução de dúvidas pontuais.

* **Modelo Utilizado:** Gemini 3.1 Pro
